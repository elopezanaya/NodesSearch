NodesSearch
===========

1) Given a tree, return all nodes that contain a given input substring (for example, input = AT)

For example:

		                        	CAT
  |                            |                         |
BAT				                    DOG		                    STORE
                     |                  |
		                KAT			          GNAT


returns nodes for [CAT, BAT, KAT, GNAT]
